# 免费版增长记忆优化记录

更新时间：2026-07-16

状态：免费版累计记忆已实现；暂不扩展付费功能。

实现日期：2026-07-16

## 产品结论

当前免费版的功能闭环已经足够：账号定位、定位找人、竞品洞察、结果导入、互动队列、AI 评分与草稿、跳转回复、插件反馈回写和本地持久化均已具备。后续不继续堆叠新模块，只对“增长记忆”做一次基础可累计优化。

## 免费版范围

- 由用户手动触发学习，不自动定时运行。
- 单账号使用，最多保留约 10 条活跃规则。
- 明显相同或高度相似的规则自动合并，不重复新增。
- 新反馈只复核相关旧规则，不把全部历史记忆交给 AI 重评。
- 每条规则保留状态、置信度、正反证据数量和最后验证时间。
- 页面显示本轮新增、合并、增强、降级和停用数量。
- 评分与草稿只注入少量高置信度活跃规则，完整历史不进入 Prompt。
- 数据继续保存在本机 SQLite；免费开源版本采用用户自备 API Key。

当前实现会为反馈生成稳定样本标识，避免同一反馈重复计数；每轮只把新反馈和最多 10 条相关旧规则交给 AI。返回结果在本地完成规则复核、证据累计、相似合并和活跃数量限制，再由用户手动确认是否应用到下一轮。

## 暂留给付费版

- 自动按批次或定期学习。
- 全量语义去重、冲突识别和规则老化提醒。
- 完整版本历史、差异查看、规则锁定与回滚。
- 多账号、跨设备同步、云备份和团队协作。
- 托管模型额度、自动周报和实验效果对比。

## 实现原则

免费版必须能够真实形成“越用越懂”的基础闭环，付费版售卖的是自动化、连续性、可控性和效果证明，而不是简单增加 AI 调用次数。开源项目中的付费能力不能只依赖前端开关；未来的 Pro 能力应主要由服务端托管功能承载。
